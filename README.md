# 网页编辑器

这是一个用于编辑网页内容的在线工具。

## 功能

- 输入网址以加载网页。
- 可编辑加载的网页内容。
- 保存编辑的内容到历史记录。
- 恢复之前保存的内容。

## 使用方法

1. 输入你想编辑的网页 URL。
2. 点击“开始编辑”按钮。
3. 修改网页内容后，点击“保存内容”以保存更改。
4. 历史记录按钮可以查看和恢复之前保存的内容。
